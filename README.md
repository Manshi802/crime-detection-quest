# 🔍 Crime Detection Quest – SQL Learning Game

An interactive gamified SQL learning platform where users solve crime mysteries using SQL queries in a detective-style storyline.

---

## 📌 About the Project

Crime Detection Quest is a browser-based educational game designed to make SQL learning fun and interactive. Players act as detectives and solve progressive crime cases using real SQL queries.

---

## 🚀 Features

- 🧠 Interactive SQL learning through gameplay
- 🔍 Detective-based mystery storyline
- ⏱️ Timer-based challenges
- 💡 Hint system for guidance
- 🏆 Score tracking system
- 📊 Live database table display
- 🎮 Multiple progressive levels
- 🎬 Cinematic intro & loading experience
- 🎯 Progress tracking bar

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Animations & UI Design)
- JavaScript (Game Logic)
- Backend API (SQL validation - optional)

---

## 🎮 How to Play

1. Read the crime story carefully
2. Write correct SQL query
3. Click "Run Query"
4. Solve all levels step by step
5. Complete the final case 🕵️

---

## 💡 Project Idea

This project is designed to make SQL learning engaging for students by combining storytelling, gamification, and real database queries.
